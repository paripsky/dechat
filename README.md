Description
---

dechat is a serverless peer to peer chat. The UI is written using React.

Develop and Build
---
to start the development server use 
```
npm run dev
```

to build use
```
npm run build
```
and deploy to any static hosting(github pages etc.)

Testing
---
```
npm run test
```
